Description

This is an example module for the SceneFlow computation.

Libraries

YARP libraries and OpenCV 2.2

Parameters

–configDisparity file.ini

  • The parameter file.ini specifies the cameras config file (e.g icubEyes.ini).

–denseFlow val

  • The parameter val specifies the flow type: dense if val=1; sparse if val=0.

–leftCamera left

  • The parameter left specifies the left camera port (e.g. /icub/camcalib/left/out).

–rightCamera right

  • The parameter right specifies the right camera port (e.g. /icub/camcalib/right/out).
